# Kulki
My version of a game called Kulki/Color lines. I've used my own implementation of A* pathfinding algorithm.

<img src="./assets/projectPreview.PNG">

## Prerequistes

You need: Node.js, npm, a browser

## Installation
```bash
npm i
```

## Running
```bash
npm start
# or
npm run start:open
```

## Building
```bash
npm run build
```

## Testing
```bash
npm test
# or
npm run test:watch
```
